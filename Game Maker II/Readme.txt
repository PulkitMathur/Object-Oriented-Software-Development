A Java Swing Application to create, play, save and load  Atari Games (Breakout, Space Invaders, Frogger, Pacman and Asteroids).

Design Patterns Used: Observer, Command, Strategy and Composite.

Technologies Used: Java, Swing, Git, GitHub, Atlassian Tools (JIRA, Confluence, Bamboo), Spring, Hibernate, Eclipse, JUnit and Mockito for testing, Log4j.
